How It Works

A peek into the sophisticated orchestration that powers every interaction

01

Prompt Enhancement

Before your query reaches the AI, it's analyzed by a secondary "utility" LLM that enriches it with context from your synced folders. This transforms vague requests into precise, contextually-aware instructions.

semantic search file tree analysis context injection
// Before enhancement
"fix the auth bug"
// After enhancement
"Review auth flow in src/auth.ts, handle OAuth callback error on line 47..."
02

Multi-Model Routing

Seline doesn't rely on a single AI. Your request is routed to specialized models: a powerful model for complex reasoning, a fast model for quick tasks, a vision model for images, and a research model for deep investigation.

Smart Tool Discovery

Instead of overwhelming the AI with 40+ tool definitions, Seline uses a "Tool Search Tool" pattern. The AI discovers only the tools it needs, when it needs them—saving thousands of tokens per request.
