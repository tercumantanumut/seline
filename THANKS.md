# Acknowledgments & Thanks

Seline is built on the shoulders of giants. We are grateful to the following open-source projects and libraries that make this application possible:

## Core AI & LLM Infrastructure
- **[Vercel AI SDK](https://sdk.vercel.ai/)** - The robust foundation for our AI interactions.
- **[Assistant UI](https://assistant-ui.com/)** - Beautiful and functional AI chat components.
- **[Ollama](https://ollama.com/)** - Empowering local LLM execution.
- **[Whisper.cpp](https://github.com/ggerganov/whisper.cpp)** - High-performance local speech-to-text.
- **[ComfyUI](https://github.com/comfyanonymous/ComfyUI)** - Our powerful backend for image generation.
- **[Hugging Face Transformers](https://huggingface.co/docs/transformers.js)** - Enabling state-of-the-art local AI models.
- **[ONNX Runtime](https://onnxruntime.ai/)** - Efficient local inference engine.
- **[Model Context Protocol (SDK)](https://modelcontextprotocol.io/)** - For seamless tool and data integration.

## Data & Search
- **[LanceDB](https://lancedb.com/)** - Our amazing embedded Vector Database for fast semantic search.
- **[Drizzle ORM](https://orm.drizzle.team/)** - Type-safe database management.
- **[Better-SQLite3](https://github.com/WiseLibs/better-sqlite3)** - High-performance SQLite implementation.
- **[Meilisearch](https://www.meilisearch.com/)** - Lightning-fast full-text search.
- **[Qdrant](https://qdrant.tech/)** - Scalable vector search engine.

## Desktop & Web Frameworks
- **[Next.js](https://nextjs.org/)** - The React framework for the web.
- **[Electron](https://www.electronjs.org/)** - Bringing Seline to the desktop.
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS for our premium styling.
- **[Radix UI](https://www.radix-ui.com/)** - Unstyled, accessible UI components.
- **[Framer Motion](https://www.framer.com/motion/)** & **[Anime.js](https://animejs.com/)** - Powering our smooth animations.
- **[Lucide Icons](https://lucide.dev/)** & **[Phosphor Icons](https://phosphoricons.com/)** - Beautiful iconography.

## Specialized Tools
- **[RTK (Rust Token Killer)](https://github.com/rtk-ai/rtk)** - The high-performance CLI proxy for token optimization.
- **[Remotion](https://www.remotion.dev/)** - Programmatic video generation.
- **[Puppeteer](https://pptr.dev/)** - Browser automation and web scraping.
- **[FFmpeg](https://ffmpeg.org/)** - The swiss army knife of multimedia processing.

## Integrations
